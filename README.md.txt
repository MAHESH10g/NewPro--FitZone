# 🏋️‍♂️ FitZone

**FitZone** is a web-based fitness platform designed to help users plan personalized gym workout routines based on their fitness level and goals.  
It provides customized exercise ideas, a simple interface, and secure data handling using MongoDB and JSP.

---

## 📘 Project Overview
FitZone simplifies the workout planning process by allowing users to:
- Create a personal account.
- Enter their basic information (age, height, weight, gender, and fitness level).
- Choose specific muscle groups.
- Get a set of customized workout suggestions dynamically generated on the webpage.

This project demonstrates full-stack web development using **HTML, CSS, JSP, and MongoDB**, showing how data-driven fitness applications can enhance user engagement and motivation.

---

## ⚙️ Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** JSP (Java Server Pages), Apache Tomcat  
- **Database:** MongoDB (NoSQL)  
- **IDE:** NetBeans  
- **Tools:** MongoDB Compass, Apache Tomcat 9.0, JDK 8+  

---

## 🚀 Features
- 🔐 User Signup & Login system  
- 🧠 Stores personal fitness details securely in MongoDB  
- 💪 Select experience level: Beginner / Intermediate / Advanced  
- 🩻 Choose muscle group (Chest, Biceps, Legs, etc.)  
- 🏃 Dynamic workout recommendations for each muscle  
- 📱 Responsive and user-friendly UI  

---

## 🧩 How to Run the Project

### 1. Prerequisites
- Install **Java JDK 8+**  
- Install **Apache Tomcat 9+**  
- Install **MongoDB Community Edition** and start the server  
- (Optional) Install **MongoDB Compass** for GUI  
- Install **NetBeans IDE**

### 2. Setup Steps
1. Start your MongoDB service (`mongod` command).  
2. Open MongoDB Compass → create a database named `gymDB`.  
3. Open the **FitZone** project in **NetBeans**.  
4. Add the **Apache Tomcat Server** under “Services”.  
5. Right-click the project → **Run**.  
6. Open your browser and visit:  
   👉 `http://localhost:8080/FitZone/index.html`
